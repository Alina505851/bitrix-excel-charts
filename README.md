# Отчёты по выгрузкам Excel

Клиентское веб-приложение на **Next.js**: загрузка файлов **`.xlsx` / `.xls`**, разбор в браузере, готовые графики по блокам **Лиды / Сделки / Контакты / Компании** (типичные выгрузки Битрикс24), фильтрация данных и выгрузка выбранных графиков в Excel.

**Данные не отправляются на сервер** — вся обработка выполняется локально в браузере.

## Требования

- **Node.js** 20+ (рекомендуется LTS)
- **npm** (или совместимый менеджер пакетов)

## Установка и запуск

```bash
npm install
npm run dev
```

Откройте в браузере: [http://localhost:3000](http://localhost:3000)

## Сборка и продакшен

```bash
npm run build
npm start
```

## Тесты и линт

```bash
npm test
npm run lint
```

## Возможности

- Загрузка Excel перетаскиванием или через диалог; ограничение размера файла указано в интерфейсе.
- Переключение **листа книги**, если в файле несколько листов.
- Включение/отключение **преднастроенных графиков** по разделам (колонки подбираются по заголовкам).
- **Фильтры** (логика «И»): несколько условий; для текста — множественный выбор значений; для чисел — диапазон мин/макс; для дат — интервал. Превью таблицы и графики учитывают фильтры.
- **Экспорт в Excel** активных графиков с учётом фильтров.

## Ограничения

- Очень большие файлы могут замедлить браузер; в приложении есть порог по числу строк на листе (см. код и подсказки в UI).
- Типы колонок **выводятся эвристически** по данным; при нестандартных выгрузках часть графиков может быть недоступна до смены файла/листа.

## Структура проекта (важное)

| Путь | Назначение |
|------|------------|
| `src/app/` | Маршруты App Router (`page.tsx`, `layout.tsx`) |
| `src/components/ReportApp.tsx` | Основной UI: загрузка, фильтры, графики, превью |
| `src/lib/excel/parseWorkbook.ts` | Чтение книги из `ArrayBuffer` |
| `src/lib/excel/exportChartsExcel.ts` | Сборка Excel с листами графиков |
| `src/lib/predefinedCharts.ts` | Спецификации графиков по сущностям |
| `src/lib/chart/buildChartData.ts` | Фильтры, агрегация, данные для Recharts |
| `src/lib/types.ts` | Типы таблицы, конфигурации графика и фильтров |
